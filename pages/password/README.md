## Content

####

### Subscription

&#x20;In the **“Title”** write the title of the subscription form. In the **“Paragraph”** write the information text. In the **“Placeholder”** you can write a text if such need arises. In the **“Button text”** you can write the name of the button.

### Banner

&#x20;Here you can select an image for the Password page, that will be displayed on the right. The subscription form will be moved to the left side.
